### Hướng dẫn cài đặt, chạy app:
Clone code về:
`git clone https://github.com/coderiukl/django-tiny-app/`

Vào thư mục: `django-tiny-app` và khởi tạo `venv`

`cd django-tiny-app`

`python -m venv venv`

**Truy cập vào môi trường**
+ Với Windows: `.venv/Scripts/Activate.ps1`
+ Mac/Linux: `source venv/bin/activate`

**Khởi động webapp**
+ Cài đặt thư viện: `pip install -r requirements.txt`.
+ Chạy lệnh:
`python manage.py makemigrations` &
`python manage.py migrate`
+ Sau đó chạy lệnh: `python manage.py runserver` và Nhấn `Ctrl + click: https://127.0.0.1/8000` để chạy trang web
